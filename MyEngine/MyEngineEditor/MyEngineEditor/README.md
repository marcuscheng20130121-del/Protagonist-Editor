# protagonist editor

This is a lightweight C# game engine editor that supports real-time code editing and compilation.


## ⚠️ Disclaimer 
**This project is currently in the ALPHA stage.** 
Please be aware that:
* **Bugs are expected:** You may encounter crashes, unexpected behavior, or memory issues.
* **UI/UX is unpolished:** The current interface is functional but not yet aesthetically refined.
* **Use at your own risk:** This software is provided "as is" for experimental purposes.

Your feedback and bug reports are welcome as I continue to improve the stability and design of the engine!

## How to compile
1. Ensure that the .NET 8.0 SDK is installed.
2. Run the following command in the terminal: `dotnet build --arch x64`
3. Run the generated .exe file.
